Detailed Description

Validating and formatting geographic coordinates.

Provides real-time validation for latitude/longitude input fields, ensuring format compliance and range validity for QR code generation.

Usage: CoordinateHelper.INSTANCE.getValidatedCoordinate(field, true)

String fr.softsf.canscan.util.CoordinateHelper.getValidatedCoordinate ( JTextField field,
boolean isLatitude )

Validates and truncates a coordinate from a JTextField for QR code generation.

Validates format -?\\d*(\\.\\d*)? and range (lat: [-90,90], lon: [-180,180]). Truncates decimals toward zero (lat: 8, lon: 7). Clears field on invalid input.

Parameters
fieldthe JTextField to validate
isLatitudetrue for latitude, false for longitude
Returns
truncated coordinate, or "" if invalid
