Detailed Description

CanScan - Swing application to generate QR Codes.

Supports two modes:

  • MECARD: generates a QR code from structured contact info (name, phone, email, etc.).
  • Free: generates a QR code from arbitrary text or URLs.

Features live preview, optional logo embedding, customizable size, colors, margin, and module style (rounded or square). Uses ZXing for QR code generation and FlatLaf for GUI styling.
