Detailed Description

CanScan — a Swing-based application for generating QR codes.

Supports two modes:

  • MECARD — generate a QR code from structured contact data (name, phone, email, etc.).
  • Free — generate a QR code from arbitrary text or URLs.

Includes live preview, optional logo embedding, adjustable size, colors, margins, and module style (rounded or square). Uses ZXing for QR generation and FlatLaf for modern UI styling.

JPanel fr.softsf.canscan.CanScan.initializeMainPanel ( )
private

Initializes and assembles the main application panel.

This method configures the QR code display area, including an overlay combining the QR label and a loading indicator. The overlay is created via createQrCodeLabelAndLoaderOverlayPanel(JProgressBar). Resize listeners are registered to ensure both the QR code and loader are dynamically resized and remain perfectly aligned when the window or components change size.

The final layout uses a BorderLayout with:

  • NORTH: header panel (titles, fields, etc.),
  • CENTER: overlay panel containing the QR code and loader centered in a transparent panel,
  • SOUTH: vertical spacer.
