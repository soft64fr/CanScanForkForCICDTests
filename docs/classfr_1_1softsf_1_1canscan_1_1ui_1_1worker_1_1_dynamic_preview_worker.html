Detailed Description

Asynchronously generates and displays a QR code preview in a Swing UI.

Preview generation runs off the Event Dispatch Thread (EDT) to maintain interface responsiveness. This class uses a debounce mechanism and background SwingWorker to avoid unnecessary regenerations when multiple input or configuration changes occur rapidly.

Each instance manages the lifecycle of a QR code preview for a specific JLabel and collaborates with a DynamicResizeWorker instance for dynamic resizing. The optional JProgressBar can show a wait/progress indicator during background processing.

Resources are properly managed: previous images are freed, background workers are cancelled, and the loader is stopped to prevent memory leaks and ensure smooth UI updates.
