Detailed Description

Base class managing asynchronous QR code tasks with a unified workflow.

Handles debounce timers, worker cancellation, resource cleanup, and error reporting. Provides lifecycle management to safely release resources during shutdown or reset.

Parameters
<T>type of result produced by the SwingWorker (e.g., ImageIcon, BufferedImage)
