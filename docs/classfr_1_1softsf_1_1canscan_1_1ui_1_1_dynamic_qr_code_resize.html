Detailed Description

Asynchronously resizes a generated QR code image for display in a Swing UI.

Resize operations are executed off the Event Dispatch Thread (EDT) to maintain UI responsiveness. This class uses a debounce mechanism and background SwingWorker to avoid unnecessary scaling when multiple layout or configuration changes occur rapidly.

Each instance manages resizing for a specific JLabel, working with a QrCodeBufferedImage as the source image. The optional JProgressBar can show a wait/progress indicator while resizing is in progress.

Resources are properly managed: previous icons are disposed, background workers are cancelled, and the loader is stopped to prevent memory leaks and ensure smooth UI updates.
